rail-net的gz文件
在alili-tool仓库下
BuildRailNetRawGz.ts生成

生成后粘贴到这里，commit push后，
手动打tag：

tag结构为 rail-net-数字

如：
git tag -a "gdcj-train-detail-2025-09-26-1" -m "gdcj-train-detail-2025-09-26-1"
git push origin "gdcj-train-detail-2025-09-26-1"

删除：
git tag -d "gdcj-train-detail-2025-09-26-1"
git push origin :refs/tags/"gdcj-train-detail-2025-09-26-1"
